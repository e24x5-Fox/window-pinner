# Window Pinner

Программа для Windows, которая позволяет «скрепить» несколько открытых окон друг с другом: перетаскиваешь одно окно — остальные окна группы плавно подъезжают следом, сохраняя своё относительное расположение.

## Возможности

- **Группировка окон.** Выбираешь любые 2+ открытых окна и связываешь их в группу через веб-интерфейс приложения.
- **Перетаскивание.** Пока тащишь одно окно группы, остальные остаются на месте; как только отпускаешь — они плавно (ease-out) подъезжают на точное правильное место. Время анимации возврата настраивается.
- **Блокировка/разблокировка группы.** Можно временно разблокировать группу (окна двигаются независимо друг от друга), переставить их как нужно, и снова заблокировать — новое расположение станет новой базой. Управлять этим можно как из панели приложения, так и прямо с экрана — у каждой активной (в фокусе) группы есть небольшой плавающий оверлей с кнопкой открепить/закрепить.
- **Синхронизация переднего плана.** При клике на любое окно группы остальные окна группы поднимаются в z-order сразу за ним, чтобы группа не терялась под другими окнами.
- **Сохранение состояния.** Группы и настройки сохраняются между перезапусками приложения.
- **Трей.** Приложение сворачивается в системный трей вместо полного закрытия по крестику; выйти можно через пункт меню в трее.
- **Своё окно.** Приложение — не браузерная вкладка, а отдельное встроенное окно (на базе WebView2/pywebview), не зависящее от того, какой браузер установлен в системе.

## Как это устроено

- Отслеживание и перемещение окон — через Win32 API (`pywin32` + `ctypes`), хуки `SetWinEventHook` для событий перемещения, фокуса и закрытия окон.
- Панель управления — локальный Flask-сервер и веб-интерфейс (HTML/CSS/JS), встроенные в native-окно приложения через `pywebview`.
- Значок в трее — через `pystray`.
- Плавающие оверлеи на экране — через `tkinter`.

## Запуск

Готовый исполняемый файл — во вкладке [Releases](../../releases). Скачай `WindowPinner.exe` и запусти, установка не требуется.

Для запуска из исходников:

```
pip install -r requirements.txt
python run.py
```

## Требования

Windows 10/11. Для отображения приложения в собственном окне (а не в браузере) нужен Microsoft Edge WebView2 Runtime — он предустановлен на подавляющем большинстве систем вместе с Edge; если его нет, приложение откроется в браузере по умолчанию.
